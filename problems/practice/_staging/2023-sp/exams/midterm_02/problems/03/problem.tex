\begin{prob}
    Consider the code below:

    \inputminted[lastline=6]{python}{\thisdir/code.py}

    What is the expected time complexity of this code if \python{A} and
    \python{B} are Python \python{set}s (hash tables) with $n$ elements each?
    State your answer as a function of $n$ using asymptotic notation.
    You may assume that the hash function used is ``good'' in that it hashes
    evenly (that is, it satisfies the simple universal hashing assumption).

    \inlineresponsebox[.75in]{$\Theta(n)$}

\end{prob}
