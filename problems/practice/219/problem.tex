\begin{prob}

    Suppose an undirected graph has 50 nodes. A full DFS is run on the graph to
    compute start and finish times. Suppose that, for each node, the difference
    between the node's start and finish is printed, and the largest difference
    is found to be 15.

    True or False: it is possible that the graph is connected.

    \tF{}

    \begin{soln}
        False
    \end{soln}

\end{prob}
