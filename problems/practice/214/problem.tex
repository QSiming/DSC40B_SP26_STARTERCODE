\begin{prob}

    True or False: a depth first search with source node $s$ is guaranteed to
    find the \emph{longest} simple path starting from $s$ in a connected,
    undirected graph $G$

    That is, suppose that there is a unique longest simple path in $G$, and call this path $p$.
    True or False: the longest simple path in the DFS forest must be also be $p$.

    \tF{}

    \begin{soln}
        False
    \end{soln}

\end{prob}
