This problem is autograded but requires manual grading for time complexity.

* (+2 points): linear expected time
* (+1 points): Theta(n log n) expected time or better

Note that an optimal solution of Theta(n) will get both of the above, for a
total of 3 points.

This problem also requires manual *spot checking*: Look at the submissions
which received zero points (or close to zero points) and determine if the
autograder is failing. Also ask yourself: should the student receive more
points than the autograder is giving? If so, we may want to change the
autograder.
