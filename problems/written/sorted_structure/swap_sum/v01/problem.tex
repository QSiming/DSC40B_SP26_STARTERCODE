\begin{prob}
    % Write the problem here.
    In a file named \python{ep04.py}, write a function named \python{swap_sum(A, B)}
    which, given two \textbf{sorted}
    integer arrays $\python{A}$ and $\python{B}$, returns a pair of elements \python{(a,b)} --
    one from $A$ and one from $B$ -- such that after swapping these elements,
    \python{sum(A) == sum(B)}. You may assume that such a pair exists.

    For example, suppose \python{A = [2,3,4,5]} and \python{B = [8,1,3,1,7]}. 
    Swapping $4$ and $7$ results in arrays $(2,5,7,3)$ and
    $(8,3,1,1,4)$; the elements of each sum to 17. Your algorithm should run in
    time $\Theta(n)$, where $n$ is the size of the larger of the two lists.
\end{prob}
